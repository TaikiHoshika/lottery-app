# くじ引きアプリ

JavaScriptとReactを使用したシンプルなくじ引きアプリです。

## 機能

- 1から指定した範囲内でランダムに数字を抽選
- 範囲指定はinputで入力可能
- 中央にくじ引き箱の画像を表示
- 抽選結果をモーダルで表示

## セットアップ

1. 依存関係のインストール:
```bash
npm install
```

2. 開発サーバーの起動:
```bash
npm start
```

3. ブラウザで `http://localhost:3000` を開く

## 使い方

1. 範囲指定の入力欄に、抽選したい最大値を入力します（デフォルトは10）
2. 「抽選する」ボタンをクリック
3. モーダルに抽選結果が表示されます
4. モーダルの「閉じる」ボタンまたは背景をクリックして閉じます

## ビルド

本番環境用にビルドする場合:
```bash
npm run build
```
